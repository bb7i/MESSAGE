#Простой TCP клиент-сервер на Python

Этот проект демонстрирует базовое сетевое взаимодействие между клиентом и сервером с использованием TCP-сокетов в Python.

#Описание

Проект состоит из двух частей:

#TCP-сервер
Ожидает подключение клиента, принимает данные и отправляет их обратно (эхо-сервер).

#TCP-клиент
Подключается к серверу, отправляет сообщение и получает ответ.
